# Agente IA Visual com Lovable e Vapi

Este projeto foca no desenvolvimento de um agente de IA visual para agendamento e vídeo animado, integrando Calendly e IA visual para comunicação.

## 🛠️ Tecnologias
- Lovable (Front-end)
- Vapi (API de voz e vídeo)
- Calendly (Agendamento)
- IA (para vídeo animado)

## 🚀 Funcionalidades
- Agendamento automatizado via Calendly.
- Geração de vídeo animado com IA para comunicação.
- Integração de voz e vídeo para interação com o usuário.
- Automação de processos de agendamento e comunicação.

## 🧪 Como usar
```bash
git clone https://github.com/Raoni022/Agente_IA_Visual_Lovable_Vapi.git
cd Agente_IA_Visual_Lovable_Vapi
pip install -r requirements.txt
# Configurar variáveis de ambiente e executar o script principal
# (Pode envolver configuração de APIs e serviços externos)
```
